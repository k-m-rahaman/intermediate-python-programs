# Intermediate Python Programming Collection

A structured collection of intermediate-level Python programs designed to strengthen problem-solving skills and deepen understanding of core Python concepts.

This repository focuses on practical programming exercises involving functions, lists, dictionaries, loops, and file handling.

These programs help bridge the gap between beginner Python programming and real-world software development.

---

## Programs Included

### 1️⃣ Guessing Game
An interactive game where the user attempts to guess a randomly generated number while receiving hints.

### 2️⃣ Grade Calculator
Calculates the average marks of multiple subjects and assigns a grade based on the result using functions.

### 3️⃣ Contact Book
A simple contact management system implemented using Python dictionaries.

### 4️⃣ Password Generator
Generates secure random passwords using letters, digits, and special characters.

### 5️⃣ Number Frequency Counter
Counts how many times each number appears in a list using dictionary-based frequency tracking.

### 6️⃣ File Word Counter
Reads a text file and counts the total number of words using file handling.
